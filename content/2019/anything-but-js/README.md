In this talk, I explain why a developer might want to reach for tools outside of JavaScript. We then cover the basics of getting started with TypeScript, ReasonML, and machine learning.

- TypeScript － Helpful for learning about typed languages and "safe" development syntax.
- ReasonML － Another avenue for learning about types, built on OCaml and therefor more comfortable for JavaScript-familiar developers.
- Machine learning － A set of tools that can be plugged into any framework you're already comfortable with, not just JavaScript but also Python, C-family languages.
